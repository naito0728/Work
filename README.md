【MySQL ログイン】
mysql -u root -P 3306 -p

【MySQL 起動】
net start mysql57

【DB】
DB名：task
テーブル名：tasks
ホスト；localhost
ユーザ：root
pw:0728

【コマンド一覧】
1:タスク一覧
2:タスク作成
3:タスク更新
4:タスク削除
5:終了


コマンド1(タスク一覧)
→ステータス指定で検索条件絞り込み
→番号存在チェック
→正常時ログ出力
→エラーハンドリング

コマンド2(タスク追加)
→タスク名、内容、担当者、期限、ステータス(番号指定)入力
→正常時ログ出力
→エラーハンドリング

コマンド3(タスク更新)
→タスク一覧表示
→表示されたタスクから変更対象のID指定
→全項目変更可能
→タスク無しは処理抜け
→ID存在チェック
→正常時ログ出力
→エラーハンドリング

コマンド4(タスク削除)
→タスク一覧表示
→指定されたIDを削除
→タスク無しは処理抜け
→ID存在チェック
→正常時ログ出力
→エラーハンドリング
