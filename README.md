【MySQL ログイン】
mysql -u root -P 3306 -p

【MySQL 起動】
net start mysql57

【MySQL 終了】
net stop mysql57

【DB】
DB名：task
テーブル名：tasks
ホスト；localhost
ユーザ：root
pw:0728

【コマンド一覧】
1:タスク一覧
2:タスク作成
3:タスク更新
4:タスク削除
5:終了

【動作確認方法】
コマンド2よりタスク作成後、コマンド1、3、4の実行を行う。
データ未取得の状態で行うと処理が終了し、コマンド選択に戻る。

コマンド1(タスク一覧)
1.ステータス1~4を指定
2.1よりデータ未取得の場合はメッセージ出力。
　存在しない番号選択はエラーメッセージ
3.データ取得の場合一覧表示
4.エラーハンドリング

コマンド2(タスク追加)
1.タスク名、内容、担当者、期限、ステータス(番号指定)入力
　型に差異がある場合、エラー出力
2.正常時ログ出力
3.エラーハンドリング

コマンド3(タスク更新)
1.タスク一覧表示(同様の動作。ただし4："すべて"は検索のみのため除く)
2.表示されたタスクから変更対象のID指定
3.全項目変更
4.ID存在チェック
5.正常時ログ出力
6.エラーハンドリング

コマンド4(タスク削除)
1.タスク一覧表示(同様の動作。ただし4："すべて"は検索のみのため除く)
2.指定のIDを削除
3.ID存在チェック
4.正常時ログ出力
5.エラーハンドリング
